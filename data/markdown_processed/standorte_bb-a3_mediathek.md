# Mediathek der Universitätsbibliothek Mannheim

Im Bibliotheksbereich A3 finden Sie eine umfangreiche Sammlung von Filmen auf DVD.

## Filme auf DVD

- DVDs können an speziell ausgestatteten Arbeitsplätzen im Bereich A3 angesehen werden.
- Hauptamtliches wissenschaftliches Personal, Promotionsstudierende sowie Gastwissenschaftlerinnen und Gastwissenschaftler können DVDs für ein Semester ausleihen.
- Studierende der Universität Mannheim können DVDs während der Hausarbeits- und Prüfungsvorbereitungszeit ausleihen, wenn sie eine [Dozentenbescheinigung](https://www.bib.uni-mannheim.de/media/Einrichtungen/Universitaetsbibliothek/Dokumente/Medien/Dozentenbescheinigung_Mediathek_Ausleihe_2023.doc) (PDF, 285 KB) vorlegen.

Die Universitätsbibliothek Mannheim ist Mitglied im [Netzwerk Mediatheken](https://www.netzwerk-mediatheken.de/).

Den gesamten Bestand der Filme auf DVD finden Sie, indem Sie in der **Erweiterten Suche** die **Zweigstelle „medio“** eingeben.
