# Linked Open Citation Database (LOC-DB)

Das Projekt [Linked Open Citation Database (LOC-DB)](https://www.bib.uni-mannheim.de/ihre-ub/projekte-der-ub/linked-open-citation-database/) wurde von 2016 bis 2018 mit Förderung der Deutschen Forschungsgemeinschaft (DFG) durchgeführt. Projektpartner sind unter anderem Prof. Dr. Andreas Dengel (DFKI Kaiserslautern), Prof. Dr. Kai Eckert (Hochschule der Medien Stuttgart) und Prof. Dr. Ansgar Scherp (ZBW – Leibniz-Informationszentrum Wirtschaft).

## Projektziel

Ziel von LOC-DB ist die Entwicklung einer Linked Open Data-Datenbank zur Erschließung von Zitationsbeziehungen in elektronischen und gedruckten Medien. Das Projekt entwickelt Werkzeuge und Prozesse, mit denen Bibliotheken an einer offenen, verteilten Infrastruktur zur Erfassung von Zitationen mitwirken können. Durch Automatisierung der Erfassungsprozesse soll ein Mehrwert für die wissenschaftliche Recherche geschaffen werden, indem Zitationsbeziehungen routinemäßig erfasst, gespeichert und im Semantic Web verfügbar gemacht werden.

Bekannte Verfahren der Informationsextraktion werden weiterentwickelt, um Literaturverzeichnisse automatisiert zu erfassen. Die Rohdaten werden mit vorhandenen Datenquellen abgeglichen und verknüpft. Zudem werden Lösungen erarbeitet, um die verknüpften Daten in bibliothekarische Präsentationssysteme einzubinden. Das System ist für den produktiven Einsatz in einzelnen Bibliotheken und für skalierbare Verbundanwendungen ausgelegt.

## Software & Daten

Alle im Projekt entwickelte Software ist öffentlich auf GitHub verfügbar: [https://github.com/locdb](https://github.com/locdb).

### Hauptkomponenten

- Backend: [loc-db](https://github.com/locdb/loc-db)
- Frontend: [locdb-frend](https://github.com/locdb/locdb-frend)
- Referenzextraktor: [automatic-reference-extractor](https://github.com/locdb/automatic-reference-extractor)

### Entstandene Daten

- Daten sind in den [OpenCitations](https://opencitations.net/) Korpus eingeflossen.

- Additional Labeled Reference Data from the LOC-DB Project (2019): [DOI 10.7801/283](https://doi.org/10.7801/283)
