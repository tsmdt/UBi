# Linked Open Citation Database (LOC-DB)

Das Projekt [Linked Open Citation Database (LOC-DB)](https://www.bib.uni-mannheim.de/ihre-ub/projekte-der-ub/linked-open-citation-database/) wurde von 2016 bis 2018 mit Förderung der Deutschen Forschungsgemeinschaft (DFG) durchgeführt. Projektpartner sind unter anderem Prof. Dr. Andreas Dengel (DFKI Kaiserslautern), Prof. Dr. Kai Eckert (Hochschule der Medien Stuttgart) und Prof. Dr. Ansgar Scherp (Deutsche Zentralbibliothek für Wirtschaftswissenschaften – ZBW).

## Zielsetzung

Im Projekt LOC-DB werden auf Basis der Linked-Data-Technologie Werkzeuge und Prozesse entwickelt, mit denen Bibliotheken an einer offenen, verteilten Infrastruktur zur Erschließung von Zitationsbeziehungen mitwirken können. Durch Automatisierung der Erfassungsprozesse soll ein Mehrwert für die wissenschaftliche Recherche geschaffen werden, indem Zitationsbeziehungen routinemäßig erfasst, gespeichert und im Semantic Web verfügbar gemacht werden. Aufwand, bearbeitete Menge und Qualität werden in einer Kosten-Nutzen-Abschätzung dokumentiert.

Das Projekt nutzt und erweitert Verfahren der Informationsextraktion zur automatisierten Erfassung beliebiger Literaturverzeichnisse aus elektronischen und gedruckten Medien. Die Rohdaten werden mit vorhandenen Datenquellen abgeglichen und verknüpft. Zudem werden Lösungen entwickelt, um die verknüpften Daten in bibliothekarische Präsentationssysteme einzubinden. Das System ist für den produktiven Einsatz in einzelnen Bibliotheken und für skalierbare Verbundanwendungen ausgelegt.

## Software & Daten

Alle im Projekt entwickelten Softwarekomponenten sind öffentlich auf GitHub verfügbar unter [https://github.com/locdb](https://github.com/locdb):

- **Backend:** [loc-db](https://github.com/locdb/loc-db)  
- **Frontend:** [locdb-frend](https://github.com/locdb/locdb-frend)  
- **Referenzextraktor:** [automatic-reference-extractor](https://github.com/locdb/automatic-reference-extractor)  

Entstandene Daten aus dem Projekt:

- Daten sind in den [OpenCitations](https://opencitations.net/) Korpus eingeflossen.  
- Labeled Reference Data from the Linked Open Citation Database (LOC-DB) Project (2018): [DOI 10.7801/268](https://doi.org/10.7801/268)  
- Additional Labeled Reference Data from the LOC-DB Project (2019): [DOI 10.7801/283](https://doi.org/10.7801/283)
